Certificados

HTML

En el curso de HTML, adquirí un sólido dominio que me permite crear estructuras web sólidas y bien organizadas, permitiendo que la información se presente de manera efectiva en un sitio web. Aprendí a diseñar páginas web atractivas y funcionales, utilizando etiquetas semánticas e indentando correctamente el código.

CSS

Después de completar el curso de CSS, soy capaz de diseñar sitios web visualmente atractivos y personalizados. Aprendí a dar estilo y formato a las páginas web, controlando el aspecto y la presentación de los elementos, lo que me permite crear sitios web con una apariencia única y atractiva.

JavaScript

En el curso de JavaScript, aprendí los fundamentos y cómo usarlos para dar vida a un sitio web. Aprendí a desarrollar componentes del front-end, lo que me permite construir un sitio web impresionante con una interfaz de usuario atractiva y funcional.

React Js

En el curso de React Js, tuve la oportunidad de adentrarme en el emocionante mundo de la creación de interfaces de usuario altamente dinámicas y eficientes. Aprendí a desarrollar aplicaciones de una sola página de manera efectiva, lo que me permite ofrecer a los usuarios una experiencia de navegación fluida y enriquecedora.
